## 运行命令

```bash
# 开发运行命令
yarn dev

# 打包命令
yarn build

# 打包后运行命令
yarn start

# 基于 pm2 的运行命令
yarn pm2

# 单元测试
yarn jest

# lint 测试
yarn lint

# 代码格式化
yarn prettier

# 一键上传代码
yarn cz
```
